# kasparro-agentic-fb-analyst-manikanta-mummidi

Unique Kasparro assignment submission prepared for Manikanta Mummidi.

## Quick start
```bash
python -m venv .venv
# Windows (PowerShell): .venv\Scripts\Activate.bat
# Or in VS Code set terminal to Command Prompt and run:
.venv\Scripts\activate
pip install -r requirements.txt
python -m src.orchestrator.run "Analyze ROAS drop in last 14 days"
```

Outputs: `reports/insights.json`, `reports/creatives.json`, `reports/report.md`.
